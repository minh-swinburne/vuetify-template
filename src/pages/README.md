# Pages

Vue components created in this folder will automatically be converted to navigatable routes.

Full documentation for this feature can be found in the Official [unplugin-vue-router](https://uvr.esm.is/introduction.html) docs.

Important notes: If you are using Vue Router 5+, the file-based routing feature is now merged into the main Vue Router package (see this [migration note](https://router.vuejs.org/guide/migration/v4-to-v5.html)), so you can use it without installing unplugin-vue-router separately. Just ensure you have Vue Router 5+ installed and configured in your project.
